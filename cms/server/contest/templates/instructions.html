{% trans %}Contest Rules{% endtrans %}

  1. {% trans %}This contest is for ALL Round 2 candidates.{% endtrans %}
  2. {% trans %}There are 100 marks in total. The marks are allocated as follows: Question 1 is worth 15 marks, Question 2 is worth 22 marks, Question 3 is worth 28 marks, and Question 4 is worth 35 marks. The number of test cases for each question is hidden, although the marks for each test case will be evenly distributed across the mark allocation.{% endtrans %}
  3. {% trans %}You have two hours (120 minutes) to attempt as many questions as possible.{% endtrans %}
  4. {% trans %}You may assume that the user input will satisfy the problem specification and so you do not need to validate the input.{% endtrans %}
  5. {% trans %}You should not write code to produce only specific answers, as the judges will use other test cases.{% endtrans %}
  6. {% trans %} Any attempt to access any other website or source of information will disqualify you, with the exception of online compilers such as{% endtrans %} rextester. {% trans %}For programming language documentation, refer to the {% endtrans %} {% trans %}Documentation{% endtrans %} {% trans %}link given on the pane on the left-hand side.{% endtrans %}
  7. {% trans %}Any submissions on the evaluator should NOT attempt any of the following:{% endtrans %}
    • {% trans %}access the internal evaluation server network,{% endtrans %}
    • {% trans %}fork or execute external programs,{% endtrans %}
    • {% trans %}create or access files,{% endtrans %}
    • {% trans %}attack the system security or the grader,{% endtrans %}
    • {% trans %}change file system permissions, or{% endtrans %}
    • {% trans %}read file system information.{% endtrans %}
    {% trans %}A competitor whose submissions attempt any of the above may be disqualified.{% endtrans %}
  8. {% trans %}Do not communicate the task statements or nature of the problems with other students from different schools until after 6:30 PM on 21 August.{% endtrans %}
  9. {% trans %}Scores per question will be sent to your school after 21 August. Scores will also be available on the online evaluator after 6:30 PM, once all submissions have been evaluated.{% endtrans %}

{% trans %}Instructions{% endtrans %}

  1. {% trans %}Verify that your full name give on the bar in the top right hand corner is correct.{% endtrans %} {% trans %}If there are errors, then{% endtrans %} {% trans %}submit a question{% endtrans %} {% trans %}with the subject heading "Full Name", and in the body of the question, simply give your correct full name.{% endtrans %}
  2. {% trans %}Once you are ready to begin the contest, click on "Start". This sets the clock (top left) running. The Second Round questions will appear on the pane on the left-hand side. {% endtrans %}
  3. {% trans %}You will now have two hours (120 minutes) to complete all 4 questions.{% endtrans %}
  4. {% trans %}Click on "Statement", underneath the first question you wish to answer. Click on either "Statement in English" or "Statement in Afrikaans", depending on which language you wish to use.{% endtrans %}
  5. {% trans %}Using the template files, or otherwise, write a program to solve the problem. Cut and paste or type the program in the template provided for your language.{% endtrans %}
  6. {% trans %}Click on "Submissions". Click "Choose File", and select the source code of your program you wish to submit.{% endtrans %}
  7. {% trans %}Select the relevant programming language as shown below, and then click "Submit".{% endtrans %}
  8. {% trans %}You will only receive feedback for the example test cases given in the task statement (to ensure that the format of your output is correct). No feedback will be given for any further test cases. The evaluator will simply display your score as "N/A" for the duration of the contest.{% endtrans %}
  9. {% trans %}For each question, your score will be the highest score obtained across all your submissions for that question. You will only be permitted to submit at most 25 submissions for each question.{% endtrans %}
  10. {% trans %}You may use the{% endtrans %} {% trans %}Communication{% endtrans %} {% trans %}(top of left pane) but note that responses may be delayed.{% endtrans %}
  11. {% trans %}Select a new question from the list on the left.{% endtrans %}
  12. {% trans %}Do not wait until the last few minutes to submit your programs.{% endtrans %}
  13. {% trans %}After 120 minutes, or at 6:30 PM, whatever comes first, the server will end your participation. Should you finish before the time, check that you have answered all the questions.{% endtrans %}
